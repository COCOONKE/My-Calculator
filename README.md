# Visual Basic Windows Forms Calculator

A fully functional, desktop standard calculator built using Visual Basic .NET and Windows Forms. This application mimics the classic Windows utility layout and handles sequential calculation tracking.

## Features
- Shared event-driven code architecture for numeric inputs and operator handling.
- Real-time expression parsing (displays full equations like `9 x 5 = 45`).
- Complete arithmetic suite: Addition, Subtraction, Multiplication, and Division.
- Edge-case handling (such as prevention of multiple decimal points and zero-division blocks).

## Technologies Used
- **Language:** Visual Basic (VB.NET)
- **Framework:** .NET Framework / Windows Forms (WinForms)
- **IDE:** Visual Studio
